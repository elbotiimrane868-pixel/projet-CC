Real Estate Sales Delay Prediction — Morocco 🇲🇦This project leverages Machine Learning and AI-driven data collection to predict the "Time-to-Sell" (Délai de Vente) for real estate listings across 12 Moroccan cities. By moving beyond ready-made datasets, this pipeline implements a custom autonomous scraping agent to build a robust database of over 12,000 listings.🚀 OverviewIn a market characterized by information asymmetry, this tool provides transparency by estimating the liquidity of a property based on 19 distinct variables. The project covers the entire data lifecycle: Autonomous Collection → Cleaning & Feature Engineering → Predictive Modeling.Key MetricsData Points: 12,259 collected listings.Transactions Analyzed: 3,057 verified data points.Geographic Scope: 12 major Moroccan cities.Performance: Best model achieved an $R^{2} = 0.76$.🛠 Tech StackData Collection: AI Scraper Agent (designed to bypass anti-bot protections).Analysis: Python (Pandas, NumPy).Machine Learning: Gradient Boosting (Scikit-Learn / XGBoost).Infrastructure: Pipeline designed for scalability and API deployment (FastAPI).📂 Pipeline Architecture1. Feature Engineering & VariablesThe model analyzes 19 variables to determine sales delays, including:Property Specs: Surface area, room count, and property type.Location: City-specific liquidity indices.Market Context: Pricing vs. local market averages.3. ModelingMultiple algorithms were tested, with Gradient Boosting emerging as the most effective for capturing non-linear relationships in the Moroccan real estate market.📈 Roadmap & PerspectivesShort Term[ ] Integration of official notary data for price validation.[ ] Geospatial enrichment (proximity to schools, transport) via OpenStreetMap API.[ ] Advanced feature engineering for seasonality.Medium Term[ ] Deployment: REST API via FastAPI.[ ] Dynamic Dashboard: A pricing simulator to estimate sale delays under various price scenarios.[ ] Price Prediction: Expanding the model to predict the final transaction price.🎓 Academic ContextDeveloped at ENCG Settat (Gestion-Finance) by Imrane El Boti and Hiba Bounacer, under the supervision of M.
